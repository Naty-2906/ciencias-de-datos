Descripción General
Este proyecto tiene como objetivo contruir y evaluar un modelo de clasificación multiclase, haciendo en una primera etapa una limpieza de datos 
con una imputacion de los dato mediante dos tecnicas. Para la segunda etapa procesamos los datos en el cual tuvimos que averiguar 
las varibales que mas varianza explicativa tenia y crear un data frame.
Para la tercera etapa clasificamos los modelo y vimos cual tenia una mejor interpretación de los modelos
